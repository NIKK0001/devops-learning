# Day 03 – Linux Package Management & Bash Basics

## Package Management (apt)
- apt is the package manager for Ubuntu/Debian systems
- Used to install, update, and remove software
- DevOps engineers rely on apt to set up servers quickly

### Common apt commands
- apt update
- apt upgrade
- apt install
- apt remove
- apt search

## Bash Scripting Basics
- Bash scripts automate multiple Linux commands
- Script files use .sh extension
- First line of a script is called shebang (#!/bin/bash)

## Why This Matters in DevOps
- Manual work does not scale
- Scripts help automate server setup and deployments
- Bash scripting is used in CI/CD pipelines and cloud init scripts
