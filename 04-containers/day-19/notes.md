# Day 19 – Docker Fundamentals (In-Depth)

## What I Learned
- Docker architecture (CLI, daemon, containers)
- Installing and running Docker on EC2
- Difference between image and container
- Running nginx container with port mapping
- Container lifecycle management

## Production Relevance
- Containers are isolated Linux processes
- Used for predictable application deployment
- Core foundation for CI/CD and Kubernetes

## Key Insight
Docker simplifies application deployment by packaging runtime, dependencies, and app together.
