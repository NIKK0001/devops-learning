# Day 20 – Docker Images & Dockerfile (In-Depth)

## What I Learned
- What Docker images are and how they are built
- Docker image layers and caching
- Dockerfile instructions (FROM, RUN, COPY, CMD)
- Building custom Docker images
- Running containers from custom images

## Production Relevance
- CI/CD pipelines build Docker images
- Layer caching improves build speed
- Dockerfiles define reproducible environments

## Key Insight
Docker images are immutable, layered templates used to create containers.
